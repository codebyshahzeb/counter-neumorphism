Counter App Neumorphism

A clean and modern neumorphism-style counter app built using HTML, CSS, and JavaScript.
This app features soft, extruded UI elements with subtle shadows, giving a tactile and realistic feel. Buttons have smooth interactions with depth effects, and the design is minimal yet visually appealing.

It is fully responsive, works on both desktop and mobile devices, and supports basic counter operations like increase, decrease, and reset.

This app is easy to customize and perfect for learning modern frontend design with a focus on UI/UX, interactive elements, and neumorphic styling.
